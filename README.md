### Description
Application in jquery and sass.
App show  weather for current location and you can put any cities.


### Built with
*JQUERY
*HTML
*SASS/CSS
*AXIOS
*GULP

### Setup

install dependencies:

$ npm install -g gulp
$ npm install --save-dev gulp
$ axios install

serve with reload:
gulp
